This project is intended to build my data pipeline skills, as well as possibly boost my resume, and emphasize geospatial data engineering skills. 

I want to pull data from two different APIs: 
One Bus Away (OBA) Rest API - Parsed JSON, arrivals, predictions, stops-for-route, etc. Higher level 
GTFS-RT - Raw protobuf binary: vehicle positions, trip updates, trip updates, alerts. Lower level, industry standard

I want to pull data, store it, and conduct analysis to determine which busses are typically late at which stops and by how much. 
Determine patterns from there, like if some routes are late at some stops consistently at some dates. 
Get more practice with industry standard tools.
